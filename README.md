# Macrograin

Dark-mode macro/calorie tracker with barcode scanning, AI food-photo recognition, and support for Indian packaged and home-cooked foods.

## Stack

- Next.js (App Router) — Vercel
- Supabase (Postgres)
- Drizzle ORM
- Auth.js (Google OAuth)

## Development

```bash
cd web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Copy `web/.env.local.example` to `web/.env.local` and fill in your own Supabase and Google OAuth credentials.
